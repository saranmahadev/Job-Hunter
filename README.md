## Development

### Prerequisites

- Python 3.10+
- `pip`

### Setup

```bash
# Clone the repository
git clone https://github.com/yourusername/interview-tracker.git
cd interview-tracker

# Install dependencies
pip install -e .[dev]

# Run the application
interview-tracker
```
